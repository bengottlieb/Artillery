Make sure you download the latest Firebase files from https://dl.google.com/firebase/sdk/ios/6_19_0/Firebase-6.19.0.zip and place them in the Firebase directory at the same level as the Xcode Project file.


Add the following library directories (from the Firebase folder):

- FirebaseAuth
- FirebaseAnalytics
- Google SignIn


Add the following Other Linker Flags in Project Settings

- ObjC
- lc++
